# Architecture

The inbox copilot has three layers:

1. Classification: map email content to business categories.
2. Operations logic: assign priority, SLA, tags, and action items.
3. Response communication: produce a response draft and daily digest.

## Future Upgrade

Connect Gmail or Outlook through Make, Zapier, n8n, or the provider API. Add an LLM to summarize long threads and rewrite replies using a brand voice.
