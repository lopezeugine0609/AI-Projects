# Usage

Run the CLI with a sample inquiry:

```powershell
python src/app.py --name "Sarah" --company "Bright Path Dental" --message "Need help with missed calls, inbox cleanup, and appointment scheduling." --weekly-hours 20 --monthly-budget 1500
```

Use the output as a portfolio demo for:

- lead qualification
- CRM-ready summaries
- discovery call preparation
- first-response drafting

The same logic can later connect to Typeform, Google Forms, GoHighLevel, Airtable, HubSpot, or an n8n webhook.
