# Contribution Guide

Thank you for contributing to the AI Sentiment Classifier project.

## How to contribute

1. Fork this repository.
2. Create a branch for your change.
3. Add tests for new features or fixes.
4. Run `pytest` to confirm the code still passes.
5. Submit a pull request with a clear description.

## Recommended improvements

- Add more training examples for better model accuracy.
- Replace the synthetic data with a real movie review dataset.
- Add a REST API or web interface.
- Improve logging and error handling.

## Coding style

- Keep modules small and focused.
- Use meaningful function and variable names.
- Document any new features in `README.md` and `docs/`.
