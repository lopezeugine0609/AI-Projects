# Contribution Guidelines

This project is a prototype for GHL / n8n lead automation connectors. Contributions can focus on:

- Replacing simulated flows with real API integrations for FieldRoutes, CallGear, and CallRail.
- Adding GoHighLevel funnel and campaign mapping logic.
- Implementing webhook handling and error recovery.
- Adding more tests around lead enrichment and duplicate matching.
- Creating a workflow orchestration layer for n8n-style node execution.
